How the web application works


The website you see here is built using HTML, CSS, and JavaScript with a Flask backend. Users will create their own account through which they can begin creating and saving maps. Creating a map is easy. Just upload as many pictures as you'd like that are taken from a smartphone with GPS tagging enabled and upload the sensor readings file recorded by the Smart Sole Android app. It's that easy. The Flask server will then create a GeoJSON file comprised of all GPS coordinates recorded by the Android app, as well as all of your images and any recorded sensor information matching the timestamps of those images. Loading this GeoJSON file about your physiological state as you took each picture.


All maps are saved by the local server and can be revisited anytime. Records of uploaded images, user accounts, and created maps are all stored in an SQL database.
